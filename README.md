# currency-converter
<h2>Sobre</h2>
<p>A funcionalidade do projeto é simples: o usuário colocar o valor que ele quer converter, de qual moeda, para qual moeda, depois de fazer o pedido e clicar no botão o valor convertido aparece para ele.</p>
